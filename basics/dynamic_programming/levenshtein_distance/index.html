–†–∞—Å—Å—Ç–æ—è–Ω–∏–µ –õ–µ–≤–µ–Ω—à—Ç–µ–π–Ω–∞ –º–µ–∂–¥—É $a_{1}, \ldots, a_{n}$ –∏ $b_{1}, \ldots, b_{m}$ –º–æ–∂–Ω–æ –≤—ã—á–∏—Å–ª–∏—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º–æ–º, –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏ –ø–æ–ª–Ω–æ—Å—Ç—å—é —Å–æ–≤–ø–∞–¥–∞—é—â–∏–º —Å –∞–ª–≥–æ—Ä–∏—Ç–º–æ–º –ø–æ–∏—Å–∫–∞ –ù–û–ü: –ø—É—Å—Ç—å $d[i, j]$ - —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –õ–µ–≤–µ–Ω—à—Ç–µ–π–Ω–∞ –º–µ–∂–¥—É –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—è–º–∏ $a_{1}, \ldots, a_{i}$ –∏ $b_{1}, \ldots, b_{j}$. –¢–æ–≥–¥–∞ $d[i, 0]=i$, $d[0, j]=j$, –∞ –ø—Ä–∏ $i, j>0$ –≤–µ—Ä–Ω–æ

$$ d[i, j]=\min \left\{\begin{array}{l} l[i-1, j]+1 \\ l[i, j-1]+1 \\ l[i-1, j-1]+\chi\left(a_{i}, b_{j}\right) \end{array}\right. $$

        if i == 0 or j == 0:

            continue
        d[i, j] = min(d[i - 1, j], d[i, j - 1]) + 1
        if a[i] == b[j]:
